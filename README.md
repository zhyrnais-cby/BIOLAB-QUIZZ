# BIOLAB-QUIZZ
BioLab Quiz aplikasi kuis edukatif yang dirancang untuk membantu siswa memahami dan menguasai materi Biologi dengan cara yang menyenangkan. Tujuannya adalah mengubah proses belajar Biologi dari menghafal menjadi sebuah misi yang harus diselesaikan dengan mengelola user profile, sistem skor, dan fitur gamifikasi seperti level, toko, dan peringkat
